Design

Design for all parts is again built with the default of MaterializeCSS library. The web brings a culture all about copying and taking inspiration from existing. Espeically with code libraries. MatieralizeCSS is a popular CSS library that builds of Google's Material Design. I used it to make the site look and feel modern.
The Bootstrap library, a default of Visual Studio, was also used (especially for the slideshows).
